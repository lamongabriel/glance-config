# Community widgets catalog

Mirror of the [Community Widgets](https://github.com/glanceapp/community-widgets)
README. The live repo is the source of truth — fetch it for the newest
additions. Custom-API widget paths are `widgets/<name>/README.md` in that repo;
extension widgets link to external repositories.
